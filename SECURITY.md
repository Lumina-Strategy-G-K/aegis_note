# Security Policy

## Reporting a Vulnerability
We take the security of **Aegis Note** very seriously. If you find any security-related issues, please **do not** open a public issue. Instead, please report it privately to:

- **Email**: [report@lumina-strategy.com]

We will acknowledge your report within 48 hours and work with you to resolve the issue as quickly as possible.

## Supported Versions
Only the latest version of Aegis Note is supported for security updates.
